# LUI 定稿：全场景案例图谱

主视觉文档：`docs/LUI定稿-全场景案例图谱.html`

## 核心定义

LUI 只承载用户与 AI 的创编对话链路：

1. 用户给 AI 的要求
2. Agent 思考中
3. Agent 思考完成
4. AI 结果说明文字
5. 阶段引导卡片

右侧用户手动编辑产生的同步、拖拽、删除、表单校验、素材数量提醒等，不进入 LUI。

## 本版覆盖的案例

### A. 正常推进类

1. 进入深度创编后的首句引导
2. 确认创意收集，生成方案规划
3. 确认方案规划，生成分镜设计
4. 确认分镜设计，生成成片设置

### B. 对话改稿类

5. 用户自然语言修改方案
6. 角色变化后，剧本大纲待同步
7. 用户点击同步剧本
8. 用户用自然语言修改分镜

### C. 右侧操作不进 LUI 类

9. 用户直接在右侧输入口播脚本
10. 用户拖动 Logo / 画中画 / 提示语
11. 删除分镜 / 添加素材数量超限 / 视频节选提醒

### D. 回改、覆盖、历史类

12. 用户回改前置步骤后点击下一步
13. 回改确认后生成新版本
14. 后续旧结果变成只读历史卡

### E. 异常与边界类

15. 生成失败与重试
16. 用户跳步骤要求直接生成视频
17. 用户问无关问题
18. 查看历史素材与下载全部素材

## 明确不进入 LUI 的文案

- 分镜一口播脚本已同步
- 口播时长变为 00:07
- 分镜修改已同步到预览
- 手动调整了品牌 Logo 位置
- 手动调整了角色画中画位置
- 已删除分镜 / 已取消空槽位
- 最多添加 3 个素材版本
- 请先处理视频节选
